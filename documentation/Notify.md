# Notify
---
### Notify
#### Notify a player

##### `Server`
##### Lib.Notify(id, message, messageType?, timeout?)

#### Parameters
- **id**: number - The player id
- **message**: string - The message to display
- **messageType?**: string - The message type
- **timeout?**: number - The timeout of the message

- **message**: string - The message to display
- **messageType?**: string - The message type
