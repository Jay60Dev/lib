# Audio
---
### Audio.Play
#### Play an audio file

##### `Client`
##### Lib.Audio.Play(file, options?)

#### Parameters
- **file**: string - The .ogg file name
- **options?**: table
    - **volume**: number
    - **looped?**: boolean
    - **onEnd?**: table
        - **file**: audioFileName
        - **options**: audioOptions
